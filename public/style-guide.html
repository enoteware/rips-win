4. UI Primitives

Button — Homepage / site CTAs (preferred)

Solid primary, no gradient. All buttons use the same radius (rounded-xl / 0.75rem). Used in HeroSection, View Full Leaderboard, CommunitySection, TectonicOfferCard.

Primary CTA Secondary CTA Large primary (uppercase)

Classes: Primary — bg-primary text-primary-foreground rounded-xl font-black glow-primary hover:scale-105.

Button — Admin / form

Same radius (rounded-xl). Only the default variant uses the logo gradient; others use solid colors. Used in admin (LeaderboardAdmin, login, SiteSettingsForm) and forms.
